PRUEBA DE 5 GRAFICOS SMARTCHARTS

1. Descomprimir este ZIP.
2. Subir a la raíz del repositorio PWA-Windows:
   - la carpeta smartcharts-test completa
   - el archivo .github/workflows/build-smartcharts-test.yml
3. Entrar en GitHub > Actions.
4. Abrir "Crear EXE prueba 5 SmartCharts".
5. Run workflow, o esperar la ejecución automática.
6. Descargar el artifact "Deriv-5-Graficos-SmartCharts-Test".

Esta versión es independiente del EXE de operaciones y sirve para probar:
- cinco gráficos simultáneos;
- diseño SmartCharts de Deriv;
- herramientas de dibujo en el gráfico activo;
- modo mosaico y ampliado.
